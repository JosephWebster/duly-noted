Run

Runs duly-typed using taking the consuing the config file (see IConfig ) found as /duly-noted.json Basic code flow is:

  1. parse the cofiguration options
  2. get the files, and pass those to the ReferenceParser
  3. output the reponse to either/both @HtmlGenerator or @MarkdownGenerator MarkdownGenerator 
                            export function run () {

Get Files from Glob

This is a simple helper to get a set of files from a glob.

                            function getFilesFromGlob(globString: string): Q.Promise<string[]> {
    return Q.Promise<string[]>((resolve, reject) => {
        glob(globString, (err, files: string[]) => {
            if (err) reject(err);
            resolve(files);
        });
    });
}
